# RivalWinner
Aplicación de torneos
Es una aplicacion con una conexion a una base de datos. La función de esta aplicación es organizar torneos de cualquier tipo en los que los usuarios se inscriben y para los 3 primeros hay premio. Los usuarios van a porder crear torneos. En la informacion del torneo se encuentra la localización, el costo de inscripción y los premios. 

Se va a usar mediante métodos y botones, y se va desarrollar en lenguaje Java.

###
Debes indicar en el Readme, al principio, de que va tu proyecto,
o sea, con un poco de detalle, para que sirve, con que lengueje y técnica.
Tienes que hacer un MVC funcional (sólo un método main estático)
Y debes implementar que tu Layout permita
que pongas un JLabel en la Cabecera con el nombre del programa,
que la ventana principal tenga nombre y
que lance una ventana de imitación de carga de unos segundos.
Además, que todos los eventos los gestione el controlador.
